# Cmdlets de mobilidade

 

_**Tópico modificado em:** 2016-12-08_

Os cmdlets de Mobilidade foram introduzidos para gerenciar o recurso de mobilidade adicionado na atualização cumulativa do Lync Server 2010: novembro de 2011. Use esses cmdlets para gerenciar configurações de recursos de mobilidade, como a configuração do Serviço de Mobilidade e as políticas de usuário.

## Cmdlets de mobilidade

Os cmdlets que configuram recursos de mobilidade permitem que você execute comandos do Shell de Gerenciamento do Lync Server ou escreva scripts para configurar e testar várias configurações de mobilidade.
