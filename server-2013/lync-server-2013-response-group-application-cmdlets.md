# Cmdlets do aplicativo do grupo de resposta

 

_**Tópico modificado em:** 2016-12-08_

O Aplicativo Grupo de Resposta é um recurso de gerenciamento de voz do Enterprise Voice que encaminha as chamadas de entrada a grupos de agentes.
