# Cmdlets de telefones e dispositivos

 

_**Tópico modificado em:** 2016-12-08_

O Microsoft Lync Server 2013 oferece vários cmdlets que permitem gerenciar telefones e outros dispositivos de hardware. Isso inclui telefones VoIP, telefones de áreas comuns (como telefones em saguão de prédio, lanchonetes ou outros locais públicos) e até mesmo telefones analógicos, telefones que não conseguem executar o Lync Phone Edition.

## Cmdlets de telefones e dispositivos

Os cmdlets do **CsDeviceUpdate** são usados para gerenciar o Serviço Web de Atualização de Dispositivos, um componente do Lync Server que permite que os administradores distribuam atualizações de firmware a telefones e outros dispositivos executando o Lync Phone Edition.
