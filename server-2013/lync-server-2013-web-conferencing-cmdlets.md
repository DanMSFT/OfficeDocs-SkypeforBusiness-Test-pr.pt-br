# Cmdlets de conferência da Web

 

_**Tópico modificado em:** 2016-12-08_

As conferências e reuniões online são elementos importantes no Microsoft Lync Server 2013. Os cmdlets CsConferencingConfiguration e CsConferencingPolicy são as principais ferramentas administrativas para o gerenciamento de conferências via Windows PowerShell.

## Cmdlets de Webconferência

Os cmdlets [New-CsClientPolicy](new-csclientpolicy.md) e [Set-CsClientPolicy](set-csclientpolicy.md) fornecem maneiras adicionais de configurar e gerenciar o Lync Server 2013.
