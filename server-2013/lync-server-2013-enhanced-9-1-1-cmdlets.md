# Cmdlets do 9-1-1 avançado

 

_**Tópico modificado em:** 2016-12-08_

O Microsoft Lync Server 2013 acompanha cmdlets que permitem implementar e gerenciar a implantação do 9-1-1 Avançado (E9-1-1) de uma solução do Enterprise Voice. Use esses cmdlets para mapear os pontos de conexão para endereços físicos e para definir as configurações necessárias para que os usuários do Enterprise Voice completem com êxito as chamadas de emergência e enviem automaticamente um local para os provedores de serviços de emergência. Não é possível configurar o E9-1-1 a partir do Painel de Controle do Lync Server. Você precisa usar cmdlets.
