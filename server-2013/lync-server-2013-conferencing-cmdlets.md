# Cmdlets de conferência

 

_**Tópico modificado em:** 2016-12-08_

O Microsoft Lync Server 2013 permite que os usuários participem de conferências de duas maneira diferentes: usando um aplicativo de conferência como o Lync 2013 ou discando em um telefone. Os usuários de discagem não podem realizar ações como exibir slides ou trocar mensagens instantâneas, mas podem participar plenamente da parte de áudio da conferência.

## Cmdlets de Conferência

Os cmdlets de **CsDialInConferencing** são usados para configurar as propriedades de discagem de conferência, incluindo desde especificar o número de telefone de usuários para chamá-los a ingressar em uma conferência até os comandos de teclado disponíveis a eles depois que ingressam em uma conferência (por exemplo, pressionar 6 para silenciar ou não o áudio de seu telefone). A maioria dos recursos de uma conferência (por exemplo, os usuários podem gravar na conferência, os usuários podem compartilhar aplicativos durante a conferência e assim por diante) são gerenciados usando os cmdlets **CsConferencingPolicy**.
