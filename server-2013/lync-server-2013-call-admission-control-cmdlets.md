# Cmdlets de controle de admissão de chamadas

 

_**Tópico modificado em:** 2016-12-08_

O controle de admissão de chamadas determina se existe largura de banda suficiente para estabelecer uma sessão de áudio ou vídeo em tempo real de qualidade aceitável. Gerencie o controle de admissão de chamadas definindo limitações e configurações para redes, locais e subredes e as interações entre eles.
